===================
 ``cloudmesh.aws``
===================

AWS plugin for Cloudmesh


==============
 Requirements
==============

1. Python 2.7
2. Pip
3. AWS configuration

   AWS Configuration files must be present.

   a. ``~/.aws/credentials``
   b. ``~/.aws/config``

   See the `AWS Docs <http://docs.aws.amazon.com/cli/latest/userguide/cli-config-files.html>`_ for details.
